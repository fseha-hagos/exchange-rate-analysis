# Exchange Rate Trend Analysis (ETB → USD)

## Overview

This project analyzes the historical trend of the Ethiopian Birr (ETB) against the US Dollar (USD) using Python, Pandas, and Matplotlib. The goal is to visualize currency trends, identify periods of high volatility, and provide insights for decision-making.

---

## Project Goals

- Explore ETB → USD exchange rate trends over time
- Visualize trends using line charts, moving averages, and distribution plots
- Identify high and low points in the currency trend
- Optional: Analyze how inflation impacts exchange rate trends
- Provide actionable insights for importers, investors, and households

---

## Data

- **Exchange Rate Data:** Historical ETB → USD rates (last 2–5 years)
- **Optional Inflation Data:** Monthly Consumer Price Index (CPI) from CSA or World Bank
- **Data Format:** CSV

---

## Technologies

- Python 3.8.10
- Pandas
- Matplotlib
- Jupyter Notebook / VS Code
